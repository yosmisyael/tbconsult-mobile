# Knowledge Base Sources

Place your TB medical guideline documents here in .md or .txt format.

Recommended sources:
- WHO TB Guidelines: https://www.who.int/publications/i/item/9789240048126
- CDC TB Guidelines: https://www.cdc.gov/tb/
- NIH TB Information: https://www.nhlbi.nih.gov/

File naming convention: `source_category_title.md`
Example: `who_guideline_symptom-screening.md`
